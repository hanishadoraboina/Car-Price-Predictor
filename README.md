# 🚗 Car Price Prediction using Machine Learning

A Machine Learning project that predicts the selling price of used cars based on features such as company, model, year, fuel type, kilometers driven, and other vehicle attributes.

The project uses the **Quikr Car Dataset** and follows a complete machine learning workflow from data preprocessing to model building and prediction.

---

## 📌 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Used Car Price Prediction

---

## 🛠️ Tech Stack

- Python
- Jupyter Notebook

- Matplotlib
- Scikit-learn

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Car-Price-Prediction-ML.git
```

Move into the project folder:

```bash
cd Car-Price-Prediction-ML
```

Install dependencies:

```bash
pip install pandas numpy matplotlib scikit-learn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
carprice.ipynb
```
